使用说明：
1.开始运行程序，即可弹出GUI对话框。要求用户输入帧数阈值和视频路径，如果用户输入的视频路径不存在，则点击“确定”按键后程序自动退出；如果文件路径存在，则点击“确认”按键程序继续运行下一步。
2.选定ROI区域。利用鼠标框定ROI区域，表示用户确定的当前界面中需要进行检测的区域，在按“Enter”键表示确认ROI区域。
3.标注机械臂。按鼠标左键，标注机械臂的两个端点，通过两点确定一条直线绘制标定的直线，再按“Enter”键确定机械臂的位置。
4.按“Enter”键之后，开始视频检测。左上角显示视频帧率，下方显示视频中检测到的机械臂运动状态，有“静止”、“向上晃动”、“向下晃动”3种状态。
5.如果想提前结束检测,则可以按“q”键结束检测。当视频播放完毕后，也会结束机械臂检测。
6.结束视频检测后，检测的视频将保存在当前文件夹下，文件名为"out.avi"。
